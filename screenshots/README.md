## Screenshot Tips

1. **Use browser screenshot tools:**
   - Chrome: F12 → Ctrl+Shift+P → "Capture screenshot"
   - Firefox: F12 → Screenshot icon
   - Or use Snipping Tool / Snagit

2. **Recommended resolution:** 1920x1080 or 1280x720

3. **File format:** PNG (better quality than JPG)

4. **File naming:** Use exact names as listed above

5. **Clean screenshots:**
   - Close unnecessary browser tabs
   - Hide bookmarks bar
   - Use incognito mode for clean browser UI

## After Adding Screenshots

1. Verify all 9 screenshots are in this folder
2. Check file names match exactly (case-sensitive)
3. Ensure images are clear and readable
4. Push to GitHub:
   ```bash
   git add screenshots/
   git commit -m "docs: add application screenshots"
   git push
   ```
